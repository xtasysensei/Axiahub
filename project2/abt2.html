About this site

This website is a demonstration site for the tutorial series "How To Build Up" on DigitalOcean's community platform. Learn how to recreate and customize this site with HTML by following the step-by-step instructions in the tutorial series. Participants will finish with an under standing of HTML and a website rready to deploy to the cloud. No coding experience is necessary!
